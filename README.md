# Stakanto

**Stakanto는 음악 퀴즈 플랫폼입니다.**

    Stkanto는 익명의 노래를 재생하고 해당 노래의 제목을 맞추는 퀴즈 플랫폼입니다.
    k-pop, pop, j-pop, game등의 노래 장르가 존재하며 매판 당 20개의 노래로 구성되어 있습니다.

---

**Stakanto의 참여 인력**

    - Client : 이동현
    - Server : 김운일
    - Mobile : 박민서
    - PM : 안도현

_Stakanto는 대덕소프트웨어마이스터고등학교 겨울방학 프로젝트입니다._
